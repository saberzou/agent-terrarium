# React Bits

**Repo:** https://github.com/DavidHDev/react-bits
**Docs:** https://reactbits.dev
**License:** MIT

## What
The largest open-source library of animated React components. 110+ free, customizable animations for text, backgrounds, and UI elements.

## Categories
- 💬 **Text Animations** — blur, typewriter, split, gradient, etc.
- 🌀 **Animations** — motion effects, transitions, reveals
- 🧩 **Components** — interactive UI elements
- 🖼️ **Backgrounds** — animated/generative backgrounds

## Key Features
- 110+ components, growing weekly
- Minimal dependencies, tree-shakeable
- Fully customizable via props or source editing
- 4 variants per component: JS-CSS, JS-TW, TS-CSS, TS-TW
- Copy-paste ready for any modern React project

## Creative Tools (Free)
- **Background Studio** — explore animated backgrounds, customize, export as video/image/code
- **Shape Magic** — inner rounded corners between shapes, export as SVG/React/clip-path
- **Texture Lab** — 20+ effects (noise, dithering, ASCII) on images/videos

## Installation
```bash
# Via shadcn
npx shadcn@latest add @react-bits/BlurText-TS-TW

# Via jsrepo
npx jsrepo add @react-bits/BlurText
```

## Use Cases
- Animated text reveals and transitions
- Interactive UI components with motion
- Generative/animated backgrounds
- Inspiration for vanilla JS/CSS animation patterns (check source for techniques)
