# ClawPal — OpenClaw Desktop GUI

- **URL:** https://clawpal.zhixian.io
- **Author:** @zhixianio
- **Source:** https://x.com/zhixianio/status/2024826407369457749
- **Platform:** macOS, Windows, Linux (free, open-source)

## Key Features
- Visual config editor for agents, channels, models (no hand-editing JSON)
- SSH support for remote OpenClaw management
- One-click backup before upgrade
- Config change history with diff preview and rollback
- Session cleanup tool (good for Discord autoThread clutter)
- Recipes — guided config patches (channel personas, dedicated agents per channel)

## When It's Useful
- Complex multi-agent Discord setups
- Frequent config changes
- Remote OpenClaw management
- Config rollback safety net
