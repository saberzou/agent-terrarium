## How to Use
Each entry shows:
- **Auth**: `No` (open), `apiKey`, `OAuth`
- **HTTPS**: Yes/No
- **CORS**: Yes/No/Unknown

## Notes
- Great starting point when prototyping — find a free API before building a scraper
- Many APIs listed are truly free; some have freemium tiers
- Check individual API docs for rate limits
- The repo itself has an API: https://github.com/davemachado/public-api
