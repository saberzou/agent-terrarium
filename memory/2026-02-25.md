### Architecture Evolution
1. Started with Three.js WebGL canvas textures for text → text looked fuzzy
2. **Rewrote to pure HTML + CSS 3D transforms** (`perspective` + `translateZ` + `rotateX`) — razor sharp native text rendering, same cylindrical scroll feel, zero dependencies
3. Scroll physics: momentum damping (0.93), elastic edge bounce, spring snap-to-center
4. Description text floats in only when project is centered and scroll settles

### Design Decisions
- **Fonts:** Sora (titles, weight 600) + PT Serif (body/descriptions)
- **Color palette:** Extracted from sixtep.com — stone bg `#F5F3EF`, warm black `#2D2A27`, accent black `#2D2A27`. Tried yellow bg (`#FFD445`) briefly but reverted.
- **Light theme** — departure from Saber's usual dark aesthetic
- Text center-aligned throughout
- No cards — just floating text (title + one-sentence description)
- No project counter
- Responsive text sizes via `clamp()` — no scaling hacks

### Key Lessons
- **WebGL text is always fuzzy.** Canvas textures are rasterized at fixed resolution. For text-heavy sites, use native HTML + CSS 3D transforms instead.
- Saber wants scroll to feel slow and deliberate on touch — `0.001` sensitivity
- Smooth scroll physics: no bounce/elastic edges, just soft resistance + gentle snap. Inspired by sixtep.com's `cubic-bezier(0.34, 1.56, 0.64, 1)` transitions.
- Animated Fluent Emoji source: `Tarikul-Islam-Anik/Animated-Fluent-Emojis` on GitHub — no elevator emoji available, used Office Building for Between Floors
