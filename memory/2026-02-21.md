## iOS lesson learned
- On iOS Safari, `touchmove` with `preventDefault()` suppresses subsequent `click` events on buttons
- Fix: bypass `preventDefault` on interactive floors, or use `touch-action: manipulation` CSS
- Always test touch interactions on actual iPhone — desktop browser won't catch this
