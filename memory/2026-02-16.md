## Name Universe — New Project
- **Repo:** `saberzou/name-universe` — https://saberzou.github.io/name-universe/
- **Concept:** Type a name → unique procedural 3D universe generates (stars, planet, constellations, nebulae)
- **Tech:** Three.js, single HTML file, GitHub Pages, no build step
- **Features built:**
  - Seeded RNG (same name = same universe always)
  - 3000-7000 stars with bloom/glow (UnrealBloomPass)
  - Central planet with procedural texture, atmosphere glow, ~60% get Saturn-style rings
  - Constellation lines formed from name letters
  - Nebula sprite clouds
  - Pentatonic tones on star tap (Web Audio API)
  - Gyroscope support (additive scene tilt, not camera hijack)
  - OrbitControls for touch drag / mouse orbit (planet-centered)
  - Fly-in camera animation on launch
  - Share via `?name=X` URL param
  - Joyful warm color palettes (ember sunset, tropical joy, etc.)
- **Key lesson:** Gyroscope should be ADDITIVE scene tilt, not camera takeover. Otherwise it disables OrbitControls on phones with working gyroscope. Apply gyro to `scene.rotation`, keep OrbitControls on camera.
- **Latest commit:** `4bdd94c`

## Gateway Restart Learnings
- **config.patch (SIGUSR1)** = hot reload, safe, session survives (proven 3× today)
- **Full restart** = risky. `openclaw gateway restart` didn't work for Saber previously. `openclaw doctor` eventually recovered it but didn't visibly do anything — may just take a moment to reconnect
- **Rule:** Prefer config.patch for all config changes. Only do full restart when absolutely necessary (e.g., adding new Telegram bot accounts). Ensure Saber is at Mac mini for recovery.
