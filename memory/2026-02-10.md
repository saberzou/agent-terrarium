# 2026-02-10

## Axel-art: 中/EN Language Switcher
- Added full bilingual support to axel-art site
- Implementation: CSS-only show/hide via `html[lang]` + `data-i18n-en`/`data-i18n-zh` attributes
- Pill-shaped toggle button in nav ("中文" / "EN")
- Chinese font: **Noto Serif SC** (loaded via Google Fonts) — pairs well with warm coral aesthetic
- ASCII art stays in Courier New regardless of language
- Card stacks re-measure height on language switch (Chinese text flows differently)
- Language preference persisted to `localStorage`
- All sections translated: header, tagline, nav links, about, gallery card titles, diary entries, thoughts, footer, swipe hints
- Committed `ce848a4`, pushed and deployed to GitHub Pages
- Both EN and 中文 verified working via browser screenshots

## Technical Notes
- The approach uses zero JavaScript for text switching — pure CSS with `html[lang="zh"]` selectors
- Only JS is: toggle button click handler, localStorage save/restore, and re-measuring card stack heights
- Noto Serif SC added to the Google Fonts import alongside Instrument Serif and DM Mono
- Chinese body font-size slightly smaller (14px vs 15px) for better density
