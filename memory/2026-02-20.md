## What happened
Major build session — took the site from buggy prototype to polished experience:

### Bugs fixed
- All story lines were stacking on top of each other (opacity fade-in but never fade-out)
- Floor counter didn't sync with content
- Scroll too sensitive on touch — one swipe jumped multiple floors
- On refresh: showed floor 0 number but floor 9 content (reversed index mapping was broken)
- Root cause of most issues: old GSAP ScrollTrigger approach wasn't suited for discrete floor navigation

### Architecture decisions
- **Ditched scroll-based navigation entirely** — now fully manual touch/wheel/keyboard handling
- One swipe = exactly one floor, `isAnimating` lock prevents queuing
- `preventDefault` on touchmove + wheel to kill native scroll
- Direct index mapping (index 0 = floor 0, no more reversed mapping)
- Container is 100vh with overflow:hidden — no actual scrolling at all

### Features added
1. **Per-floor gradients** → then upgraded to **Grainient WebGL shader** (ported from react-bits/OGL)
   - Smooth color lerp between floors (~3% per frame)
   - Animated grain texture, warping noise
2. **Doto font** for floor number (dot-matrix elevator display, `font-variation-settings: "ROND" 0`)
3. **All text switched to Sora** (dropped Crimson Text serif)
4. **EN/CN language toggle** — global via localStorage, controls all pages
5. **Info page** (info.html) — fully localized, building specs, story list, Aria quote
6. **Sparkle icon** for info button (replaced "i")
7. **Wider content area** (900px) for longer story text
8. **Mobile responsive** — smaller fonts (15-16px), 30px padding, repositioned story lines
