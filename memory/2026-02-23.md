## JSON Prompts — New Project

Built **JSON Prompts** (jsonprompts.co vibes, "Jason Bourne" pun) — a curated gallery of AI image generation prompts in JSON format.

- **Repo:** https://github.com/saberzou/json-prompts
- **Live:** https://saberzou.github.io/json-prompts/
- **Inspiration:** https://extraordinarythings.co/ — studied its grid architecture (3×3 tiled copies, pannable canvas, overflow:hidden viewport, mixed tile sizes)

### Architecture
- Single `index.html`, no build step
- `prompts.json` data file — easy to add entries
- **Grid view:** Infinite pannable canvas (drag/scroll), 3×3 tiled copies for seamless wrapping, 1px `rgba(255,255,255,0.10)` gaps, mixed tile sizes (sq/landscape/portrait in 8-col grid). Cell size: 220px base, 160px mobile, 280px desktop.
- **List view:** Stacked carousel — center image large, above/below smaller (55% scale), smooth CSS transitions, wheel/swipe/arrow navigation
- **Modal:** Full image + JSON prompt + one-click copy + source link + tags
- Fonts: Archivo Black + DM Sans + JetBrains Mono, dark void aesthetic

### Lessons
- X Articles embed JSON in `entityMap[].value.data.markdown` — FxTwitter API returns full article structure including code blocks
- Notion S3 URLs expire in 1hr — download immediately
- Grid tile fill: simple `i % length` cycling causes ugly repetition with few prompts. Fixed with adjacency-aware random fill (check left, top, diagonals → pick non-duplicate)
